\noindent This has been a reasonable assumtion before the the \lhcb measurement which showed that the measured \phiS{eff} value is consistent with the Standard
Model prediction \phiS{SM,tree}, within the current experimental accuracy. However with increasing accuracy the last assumption needs to be revisited.
Specifically, because of the fact that penguin topology contributions shift the measured value of \phiS{eff}.
The last shift could be misinterpreted as $\Delta\phiS{NP}$ if the penguin contributions are not properly esimated.
Thus in order to correctly probe New Physics contributions in fututre measurements the penguin topology contributions $\Delta\phiS{SM,penguin}$
have to be estimated.

\begin{equation}
\phis^{\text {eff}} = \phis^{\tiny \text{SM,tree}} + \Delta\phiS{SM, penguin} + \Delta\phis^{\tiny \text{NP}}
 \label{phis_sm_peng}
\end{equation}
